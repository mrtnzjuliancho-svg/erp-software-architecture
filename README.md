# Taller de Arquitectura de Software - ERP

Este repositorio contiene el desarrollo del taller de arquitectura de software,
cuyo objetivo es aplicar prácticas de gestión ágil, diseño arquitectónico y
documentación utilizando la plantilla arc42.

El sistema analizado corresponde a un ERP, con énfasis en el Módulo de Compras.
